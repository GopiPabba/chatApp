Instructions on how to compile your service?
used express js to compile the service.
localhost:3000/ to post chat data
localhost:3000/chat post data
localhost:3000/chat/1 get chat using chat id
localhost:3000/chats/gopi get chat using username

Instructions on how to run your service locally?
Used node server to run the application on the local system. Configured the application to use port 3000.
